Custom Heap Manager

Custom implementations of the implicit and explicit heap managers in C. The implicit manager uses block headers and in-place reallocation, whereas the explicit uses a doubly linked list to keep track of free memory blocks on the heap and coalescing for better utilization.

Visualizing StackOverflow Dataset

What makes a good StackOverflow question post? We shed light to this by writing SQL to query data from StackOverflow's BigQuery public database of size 208 GB, created visualizations on queried tables, and made predictions with engineered features and logistic regression models.
